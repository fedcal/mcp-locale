# MCP Protocol

Creazione di un mcp server e di un mcp client seguendo la guida presente nel sito [Model Context Protocol](https://modelcontextprotocol.io/docs/getting-started/intro). L'intento è quello di 
sperimentare la creazione e l'utilizzo su un personal computer.

Verrà creato principalmente un server con delle funzioni volte ad aiutare una persona nelle attività quotidiane o nell'organizzazione di piccoli eventi ricorrenti che hanno una struttura ben 
definita, per poi procedere allo sviluppo di piccole funzionalità rivolte alle piccole e medie imprese.

In questi ultimi due casi ci si scontra spesso con attrezzature informatiche un po' datate e non tutti sono disposti ad aggiornarsi per cogliere tutte le potenzialità delle nuove tecnologie.
Infatti l'obiettivo sarà anche quello di sperimentare l'uso del server in un cloud per facilitare la computazione.
